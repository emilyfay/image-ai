Image Completion with Deep Learning

The model was trained using a deep convolutional generative adversarial network (DCGAN).
The code is written in Tensorflow, using the tutorial by Brandon Amos as a starting point. His code is based on the paper "Semantic Image Inpainting with Perceptual and Contextual Losses" by Raymond Yeh et al. The method used here to infill images is modeled after the generator described by C. Sauer et al. with some slight modifications
